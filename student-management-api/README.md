# Student Management REST API
A production-ready, cloud-deployed REST API for managing student records — built with Python Flask, Supabase (PostgreSQL), and deployed on Render.
# Live Demo
https://student-management-rest-api-7c1h.onrender.com
 # Overview
The Student Management REST API is a fully functional backend service that allows users to perform Create, Read, Update, and Delete (CRUD) operations on student records. It is built with a serverless-first approach using Python Flask and powered by a cloud-hosted PostgreSQL database via Supabase.
